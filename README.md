# 🛠️ JavaScript Functional Tool: smart-login system

A dynamic web utility focused on DOM manipulation and data persistence using pure JavaScript.

## 🚀 Key Features
* **CRUD Operations:** Create, Read, and Delete data dynamically.
* **Local Storage Integration:** Your data stays saved even after refreshing the page.
* **Input Validation:** Custom logic to ensure data integrity (e.g., Regex for URLs/Emails).

## 💻 Tech Stack
* **Vanilla JavaScript** (ES6+)
* **CSS3** (Animations & Transitions)
* **HTML5 Forms**
